## Gotchas banked

- **LWC synthetic shadow mangles element `id`s** — Playwright/DOM harness must target by class/structure,
  not `#id` (class selectors like `.cg-grid` work, ids don't).
- `AuraHandledException.getMessage()` returns "Script-thrown exception" in test context — assert the type.
- Container query can NEVER style its own container (put queried classes on a descendant).
- The `sf data query` command fails in the Bash tool (Windows path-with-spaces) — use the PowerShell tool.
